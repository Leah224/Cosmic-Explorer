Capstone Project Ideas
Welcome! This repository contains my initial ideas for my UMASS Global/Springboard Capstone project.
Each idea combines some of my interests mixed with web development while challenging me to build a full-stack application.

💭 Idea 1: Mental Health Journal & Mood Tracker
Overview: I’ve always thought mental health was extremely important, and this application could help people track and maintain their mental health.

Problem: Tracking mood and mental health can be difficult without a structured system. Many people want a private space to reflect on their feelings and track trends over time.

Data: External APIs like Twinword Sentiment Analysis or Google Cloud Natural Language API could analyze journal entries to detect emotions.

Solution: Users can:

Write daily journal entries
Track mood trends through interactive charts
Set personal goals (like better sleep, mindfulness, or study habits)
Bonus Feature: Suggest coping strategies based on user mood trends.

🧬 Idea 2: Anatomy Study Helper
Overview: Since my sister and I are really into medicine, I think making a tool to study anatomy would be both useful and fun. People could look up organs or body systems and see diagrams and key facts. I’d like to include quizzes or flashcards so you can test yourself, and maybe even a drag-and-drop labeling game. I think this would be a great way to mix learning with interactive web features.

Problem: Learning anatomy and biology can be overwhelming without interactive tools and visuals. Many students need a way to study efficiently and track their progress.

Data: APIs like the Human Anatomy API or BioPortal provide organ diagrams, system descriptions, and terminology.

Solution: Users can:

Search for organs or systems and view diagrams + key facts
Create personalized quizzes or flashcards
Track progress and highlight challenging topics
Use interactive labeling games (e.g., drag-and-drop labels on the heart or brain)
🔍 Idea 3: True Crime Case Explorer
**Overview:**For anyone interested in criminal psychology, it’d be fun to explore real cases and crime stats in one place. People could browse cases, see trends over time, and save cases they want to study more closely. I might add a section analyzing behavior patterns or motives to give it a deeper twist.

Problem: Criminal psychology enthusiasts often lack a centralized, interactive way to explore real cases and crime statistics.

Data: APIs like the FBI Crime Data API or OpenCrime provide crime statistics and case data.

Solution: Users can:

Explore crime data by state or city
Access details about notable cases
Save favorite cases for research
Visualize crime trends over time
Bonus Feature: Include a “Criminal Psychology Insights” section analyzing behavior patterns, motives, and profiling techniques.

🌌 Idea 4: Solar System Explorer & Astronomy Dashboard (Highlighted Idea/Top Choice)
Overview I’ve always loved astronomy, so this idea is about letting people explore space online. You could click on planets, moons, or asteroids and see info like size, orbit, or images. I’d also like to include live positions for satellites or planets and maybe a simulation mode for planetary alignments. It would be educational but also really fun to interact with.

Problem: Many people are curious about space but struggle to visualize the scale, position, and motion of planets, stars, and other celestial bodies.

Data:

NASA Open APIs for planet data, imagery, and astronomy pictures
Open Notify API for live ISS location data
Solution: Users can:

Explore the solar system with interactive 2D or 3D models
Click on planets, moons, or asteroids to see detailed info (size, orbit, distance from Sun, images)
Track real-time positions of satellites or planets
View historical astronomical events (e.g., eclipses, notable discoveries)
Extra Feature:

“Astronomer Mode” to simulate planetary alignments or orbits
Daily astronomy facts or images pulled from NASA APIs
Why I’m Excited About This: This project aligns with my interests and lets me combine educational tools with full-stack web development skills.

⭐ Highlighted Idea
I’m most excited about The Solar System Explorer idea because it combines my passion for space with web development. It would let me create a tool that is both educational and interactive, showcasing my skills in full-stack development while solving a real-world problem for students or enthusiasts.
